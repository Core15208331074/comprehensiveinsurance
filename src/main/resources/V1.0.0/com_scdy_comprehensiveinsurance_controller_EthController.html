添加eth传感器

请求URL

eth/saveEth POST

请求参数 application/x-www-form-urlencoded

参数名 类型 必须 描述
id string 否
createTime date 否 创建时间
updateTime date 否 更新时间
isDeleted int 否 0 表示未删除,1 表示删除,默认值为0
ethName string 否 eth名称
targetIp string 否 目标ip
targetPort string 否 目标端口
localPort string 否 本机端口
sensorModel string 否 传感器型号
deviceAddress string 否 设备地址
installPostion string 否 安装位置
tag string 否 标签，主要是传1，2，3...表示序号
isStart boolean 否 启动状态

返回结果

{
	"success":"boolean",
	"code":"string",
	"msg":"string",
	"data":{}
}
