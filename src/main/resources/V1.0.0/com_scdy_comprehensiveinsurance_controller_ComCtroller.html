请求参数 application/x-www-form-urlencoded

参数名 类型 必须 描述
id string 否 串口数据行id
pageIndex int 否 页码
pageSize int 否 每页显示多少条
startTime date 否 开始时间
endTime date 否 结束时间

返回结果

{
	"success":"boolean",
	"code":"string",
	"msg":"string",
	"data":{}
}

添加com传感器

请求URL

/com/comDataAdd POST

请求体 application/json

{
	"id":"string",
	"createTime":"date //创建时间",
	"updateTime":"date //更新时间",
	"isDeleted":"int //0 表示未删除,1 表示删除,默认值为0",
	"serialPortName":"string //串口名",
	"baudRate":"int //波特率",
	"dataBit":"string //数据位",
	"stopBit":"string //停止位",
	"checkDigit":"string //校验位",
	"sensorModel":"string //传感器型号",
	"deviceAddress":"string //设备地址",
	"installPostion":"string //安装位置",
	"tag":"string //标签，主要是传1，2，3...表示序号",
	"isStart":"boolean //启动状态"
}

返回结果

{
	"success":"boolean",
	"code":"string",
	"msg":"string",
	"data":{}
}
